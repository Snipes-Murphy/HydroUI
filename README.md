# HydroUI
A lightweight CSS library.

## Quick Start
Add this into your head for a quick integration.
```html
<link rel="stylesheet" type="text/css" href="https://vaporizestudios.github.io/HydroUI/hydroui.css">
```

It should look like this:
```html
<head>
	<link rel="stylesheet" type="text/css" href="https://vaporizestudios.github.io/HydroUI/hydroui.css">
</head>
```
